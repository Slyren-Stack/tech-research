# Software 2.0

## Summary
An essay reframing neural networks as a new kind of software: instead of writing explicit code, you curate datasets and let optimization write the program in weight-space. It reshapes how to think about building with ML.

## Key takeaways
- In 'Software 2.0', datasets and optimization replace hand-written logic.
- The job shifts from writing code to shaping data and objectives.
- Has profound implications for tooling, testing, and how teams are structured.

## Why it matters
A clarifying mental model for anyone building AI products — it changes what 'programming' means.
